# anagram_js

Finds anagrams from the input array and creates corresponding child arrays for the anagrams in the result array. 

Example input: ['rope', 'pore', 'repo', 'red rum', 'murder', 'listen', 'silent', 'endeavour']
The output should be:
[
  ['rope', 'pore', 'repo'],
  ['red rum', 'murder'],
  ['listen', 'silent'],
  ['endeavour']
]
